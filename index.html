Jasp-linux-install

JASP installation script for Debian & Ubuntu Linux

View the Project on GitHub lindemann09/jasp-linux-install

Install JASP on Debian/Ubuntu 14.10+

To install JASP 0.6.6 just download this script and execute it as sudo. Like this:

wget -P /tmp https://raw.githubusercontent.com/lindemann09/jasp-linux-install/master/install_jasp.sh
sudo sh /tmp/install_jasp.sh
