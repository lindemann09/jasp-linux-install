Install JASP 0.6.6 on Debian/Ubuntu

To install JASP 0.6.6 on Debian/Ubuntu 14.10+ just download this script and execute it as sudo. Like this:

wget -P /tmp https://raw.githubusercontent.com/lindemann09/jasp-linux-install/v0.6.6/install_jasp.sh
sudo bash /tmp/install_jasp.sh
